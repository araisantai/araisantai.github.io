A media library becomes valuable when I can find things again. Storage is cheap compared with the time lost to messy folders and unclear backups.

My current maintenance checklist:

- Import files into dated folders.
- Rename only when the default name is not useful.
- Delete obvious duplicates and failed captures early.
- Keep one working copy and one backup copy.
- Add lightweight notes for important projects or events.

The system should not require perfect tagging. A reliable folder structure plus occasional cleanup is enough for most personal media.

The next improvement is a quarterly archive review so old files do not silently become unmanaged.
